## Installation
From source:
```sh
git clone https://github.com/idan22moral/remy
cd remy
go install
```
From release:
```sh
wget https://github.com/idan22moral/remy/releases/download/v1.0.0/remy-v1.0.0-linux.tar.gz
tar -xzvf remy-v1.0.0-linux.tar.gz
mv linux/remy /usr/local/bin/remy
```
